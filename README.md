# tiny-blockchain


### To Build the C implementation

`git clone https://github.com/Cooptocracy/tiny-blockchain`

`cd tiny-blockchain/C`

`make && ./client.out`

### To Build the TS implementation

`git clone https://github.com/Cooptocracy/tiny-blockchain`

`cd tiny-blockchain/TS`

`npm install`

`npm start`

`tsc main.ts`

`node main.js`
